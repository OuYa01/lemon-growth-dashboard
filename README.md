# Station El Guerdane Dashboard

A web dashboard for monitoring citrus growth at Station El Guerdane.

## Project Structure
├── server.py ← Flask backend serving data and dashboard
├── requirements.txt ← Python dependencies
├── data/
│ └── lemon_measurements.csv ← sample or real measurement data
├── templates/
│ └── index.html ← main dashboard HTML
└── static/
├── style.css ← CSS for styling
└── dashboard.js ← JavaScript for charts and interactivity
